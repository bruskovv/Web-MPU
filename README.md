# Лабораторные работы по веб-разработке — Московский Политех

Этот репозиторий предназначен для хранения лабораторных работ по веб-разработке в рамках обучения в Московском политехническом университете.  
Здесь собраны решения практических заданий по HTML, CSS, основам вёрстки, а также другим темам курса.

## Структура проекта

- Каждая лабораторная работа находится в отдельной папке (например, `lab1`, `lab2` и т. д.).
- Внутри каждой папки содержатся все необходимые файлы: исходники HTML/CSS, изображения, а также дополнительные материалы (если требуется).
- Все задания подписаны и снабжены кратким описанием для самопроверки.

## О проекте

В этом репозитории:
- Каждая работа выполнена и структурирована максимально понятно — можно быстро найти нужное задание.
- Все стили, картинки и другие ресурсы разнесены по папкам для порядка.
- К каждой работе прилагается комментарий или краткое описание.
- **Для отображения сайта открой нужный HTML-файл в браузере.**

## Дополнительно

- Репозиторий регулярно пополняется по мере продвижения курса.
